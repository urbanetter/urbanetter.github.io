Loading process

Key to frontend performance

is to understand the page loading process

In real world, there are a lot more files

This means

  • It took almost 1 second to fetch the initial HTML
  • It took almost 3.5 seconds to reach DOM ready
  • It took almost 4.5 seconds to reach "loaded" event

The browser (IE9 in this case)

only starts 4 parallel requests for images

In older browser this restriction is set to 2

This restriction is part of the HTTP standard
